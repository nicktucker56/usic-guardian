# The USIC Guardian Mathematical Framework

## The Receptivity-Collapse Manifold

The system maps human semantic states onto a two-dimensional phenomenological manifold M defined as:

M = {(r, c) ∈ ℝ² | 0 ≤ r ≤ 1, 0 ≤ c ≤ 1, r + c ≤ 1}

where r represents receptivity and c represents collapse risk.
